# Ящик мумии

Статичный сайт-рулетка в стиле игрового ящика с мумией. Предметы нарезаны из переданных скриншотов, прокрут бесплатный, за каждый прокрут начисляется 540 очков, а выпавшие предметы сохраняются в инвентарь через `localStorage`.

## Запуск локально

```bash
pnpm install
pnpm dev
```

## Сборка для GitHub Pages

```bash
pnpm build
```

После сборки папку `dist` можно публиковать на GitHub Pages. В `vite.config.js` стоит `base: './'`, поэтому сайт работает и в подпапке репозитория.
